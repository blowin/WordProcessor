# WordProcessor

Support language:
- English
- Russian

Algorithms:
- Shuffle letter
    > 'cat' -> 'atc'

- Replace letter 
  
    >'hello' -> 'he_lo'
    
- Shuffle translate
  
  >**input:**    
  >'cat' - 'кот'    
  >'home' - 'дом'
    
    
  >**result:** 
  >'cat' - 'дом'    
  >'home' -'кот'
  `

Separators:
- New Line
- Custom
